# Markdown to PDF Converter

Простое веб-приложение для преобразования Markdown-текста в PDF-документы. Поддерживает вставку блок-схем, настройку шрифтов, размера текста и имени выходного файла. Создано для удобной работы с заметками и документацией.

## Основные возможности
- Преобразование Markdown в PDF с предварительным просмотром.
- Поддержка блок-схем через синтаксис DOT (Graphviz).
- Выбор шрифта для текста и кода (включая загрузку пользовательских шрифтов .ttf/.otf).
- Настройка размера шрифта (от 10 до 30 px).
- Генерация PDF с учетом разрывов страниц и адаптивной версткой.
- Интерактивный фон с частицами.

## Использование
1. Вставьте Markdown-текст в левое поле.
2. Настройте шрифты, размер текста и имя файла в панели управления.
3. Нажмите "Скачать PDF" для генерации документа.
